# chatbot

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install.

```bash
pip install -r requirements.txt
```

## Before Run

Can change intents.json file as per requirement

Run command

```bash

python train.py

```

## Start chatbot

Run command

```bash
python chat.py
```
